一、更新系統

sudo apt update
sudo apt upgrade -y

二、安裝 Tailscale

curl -fsSL https://tailscale.com/install.sh | sh

或使用官方教學 (apt repo)

sudo apt install apt-transport-https -y
curl -fsSL https://pkgs.tailscale.com/stable/raspbian/bookworm.gpg | sudo tee /usr/share/keyrings/tailscale-archive-keyring.gpg > /dev/null
echo "deb [signed-by=/usr/share/keyrings/tailscale-archive-keyring.gpg] https://pkgs.tailscale.com/stable/raspbian bookworm main" | sudo tee /etc/apt/sources.list.d/tailscale.list
sudo apt update
sudo apt install tailscale -y

三、啟動 tailscaled 服務

sudo systemctl enable tailscaled
sudo systemctl start tailscaled

四、登入 Tailscale VPN

sudo tailscale up

(請在齊顯的網頁上授權)

五、啟用 Subnet Router (完整 LAN 推廣)

sudo tailscale up --advertise-routes=192.168.X.0/24

(請修改 192.168.X.0/24 為自己網域範圍)

六、在 Tailscale Admin Console 認證路徑 (Approve Routes)

  1. 登入 https://login.tailscale.com/admin/machines
  2. 找到該自身 Raspberry Pi 設備
  3. 授權它的 Subnet Route

七、檢查狀態

tailscale status
tailscale ip -4

附錄、指令操作說明

  • tailscale up = 啟動 Tailscale VPN 連線
  • --advertise-routes=192.168.x.0/24 = 告知全域式 LAN 範圍
  • tailscaled = Tailscale 執行服務 (確保啟動時自動連線)

備註、如需啟動 MagicDNS 可附加指令

sudo tailscale up --advertise-routes=192.168.X.0/24 --advertise-exit-node --accept-dns

(據需選擇)
