1. 固定 Gbe_1 的 IP

  • 打開 控制台 → 網路與共用中心 → 變更介面卡設定
  • 找到 Gbe_1（90:E2:BA:A4:77:76），右鍵 → 內容 → TCP/IPv4 → 內容
  • 手動設定： IP 位址：192.168.2.100
    子網路遮罩：255.255.255.0
    預設閘道：空白
    DNS 伺服器：可以填 8.8.8.8 或留空
  • ⚡ 註：不設定閘道是避免 Windows 把整個網路預設走這張卡，讓它只用來通 RTSP。

2. 查看網卡編號

  • 打開命令提示字元 (CMD)，輸入：route print
  • 確認 Gbe_1 是介面編號 3。

3. 新增 RTSP 指定路由

  • 在 CMD（以系統管理員身分執行）輸入：
    • route -p add 192.168.2.15 mask 255.255.255.255 192.168.2.100 IF 3
  • 說明：
    • 192.168.2.15 是攝影機 IP
    • 255.255.255.255 是針對單一 IP
    • 192.168.2.100 是本地 IP
    • IF 3 是網卡介面編號
    • -p 表示永久保存

4. 測試路由是否生效

  • 在 CMD 測試：
    • tracert 192.168.2.15
  • 如果只跳一次（直接到 192.168.2.15），表示成功！

5. 附加說明

  • 新增其他 IP 時，重複新增 route -p add 指令即可。
  • 刪除路由指令：route delete 192.168.2.15
  • 每台攝影機一條路由。
  • 建議其他網卡（管理用 Gbe_2）設好閘道，避免混線。

📌 總結快速版

  • Gbe_1 固定 IP：192.168.2.100/24，不設定閘道
  • RTSP 流量綁定：route -p add 192.168.2.15 mask 255.255.255.255 192.168.2.100 IF 3
  • 測試：tracert 192.168.2.15
